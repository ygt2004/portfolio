Projects

Selected work

Polymarket Scoring Bot

Python script that fetches live data from the Polymarket API, scores every open binary market, and outputs a ranked list with suggested position sizes.

  • Pulls prices, volume, and closing dates directly from the API
  • Scores markets on four signals: volume trend, price drift, time decay, and spread width
  • Applies Kelly criterion for position sizing; exports CSV with a kill-switch on excess drawdown
→ Built from scratch — API docs read independently, no templates used.

Binance Testnet Trading Bot

Python bot monitoring three crypto pairs on the Binance Testnet, placing simulated orders based on aligned technical signals.

  • Fetches hourly candlestick data for BTC, ETH, and BNB via the Testnet API
  • Calculates RSI, Bollinger Bands, MACD, and volume — triggers only when three align
  • Logs each simulated trade with entry price, signal reason, and running P&L
→ Testnet only — built to understand how indicators interact across multiple assets.
