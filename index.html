Projects

Selected work

Polymarket Signal Engine

Surfaces mispriced binary prediction markets by scoring live data across four independent signals.

  • Pulls real-time prices, volume, and close dates from the Polymarket API
  • Scores each market on volume trend, price drift, time decay, and spread width
  • Outputs a ranked list with position sizes calculated via Kelly criterion
→ Shows how structured signal analysis can surface decisions that raw data alone doesn't make obvious.

Multi-Signal Trading Monitor

Identifies technical alignment across multiple assets before triggering any action — built to study how indicators behave together, not in isolation.

  • Tracks hourly candlestick data for three crypto pairs via Binance Testnet API
  • Requires RSI, Bollinger Bands, MACD, and volume to align before signalling
  • Logs each decision with entry price, signal reason, and running outcome
→ The insight was about convergence: single indicators are noise. Agreement between them is signal.
